# TxtToPDF and Txt Merger
### by **Srdjan Lopez LLC**

**TxtToPDF and Txt Merger** is a Windows desktop utility that lets you merge multiple `.txt` files or convert text documents into professionally formatted PDF files.

---

## Features
- Merge multiple `.txt` files into one
- Convert `.txt` to `.pdf` using [PDFsharp](https://pdfsharp.net)
- Optional page numbers and file separators
- Adjustable font size and margins
- Choose your output folder
- Drag & drop support
- Optimized async processing with progress bar

---

## Tech Stack
- **Language:** C#
- **Framework:** .NET Framework 4.8
- **UI:** Windows Forms
- **PDF Engine:** PDFsharp GDI
- **IDE:** Visual Studio 2022
